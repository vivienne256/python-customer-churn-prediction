# Customer Churn Prediction with Machine Learning

## Project Overview

This project uses Python and machine learning to predict customer churn based on customer demographics, account information, subscribed services, and billing details.

The goal is to build and evaluate classification models that can help identify customers at higher risk of churn.

## Business Problem

Customer churn reduces recurring revenue and increases customer acquisition costs.  
By predicting churn risk, a company can prioritize retention actions for customers who are more likely to leave.

## Dataset

This project uses the Telco Customer Churn dataset.

The dataset includes:
- Customer demographic information
- Account tenure
- Subscribed services
- Contract type
- Payment method
- Monthly charges
- Churn status

## Key Business Focus

For churn prediction, recall is especially important because the business wants to identify as many potential churn customers as possible.
